# CHAMPFIT — Premium Fitness Coaching Portfolio

A high-end, production-ready transformation portfolio and client intake platform engineered for online fitness coaches and personal trainers. Designed with a mobile-first philosophy to convert social media traffic (Instagram Bio) into high-paying coaching clients.

## ⚡ Luxury Aesthetic & Features
- **Premium Dark UI:** Designed with a striking obsidian, titanium white, and brushed champagne gold color palette inspired by elite fitness brands like Gymshark and CBum Fitness.
- **Interactive Before/After Slider:** An ultra-smooth, responsive custom image comparison slider with an overlay dynamic lightbox viewer.
- **Coach Workspace Dashboard:** Built-in hidden workspace console to seamlessly add, update, or clear client transformation profiles and testimonials in real-time (powered by persistent `localStorage`).
- **High-Conversion Intake Funnel:** An automated structural assessment form that dynamically sanitizes client data (age, weight, primary metrics) and routes it directly into an incoming WhatsApp onboarding pipeline.
- **Performance Optimized:** 100% fluid, responsive design animated via Framer Motion, utilizing lazy-loaded assets and modern SVG elements to minimize layout shifts.

## 🛠️ Tech Stack
- **Framework:** React.js (Vite)
- **Styling:** Tailwind CSS (Custom Premium Extended Palette)
- **Animations:** Framer Motion
- **Icons:** Lucide React & Custom Optimized SVGs
- **Data Persistence:** Client-side HTML5 LocalStorage
