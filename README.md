# CHAMPFIT — Premium Fitness Coaching Portfolio

A high-end, production-ready transformation portfolio and client intake platform engineered for online fitness coaches and personal trainers. Designed with a mobile-first philosophy to convert social media traffic (Instagram Bio) into high-paying coaching clients.

## ⚡ Luxury Aesthetic & Features

- Premium Dark UI with obsidian, titanium white, and brushed champagne gold color palette.
- Interactive Before/After Slider with responsive image comparison.
- Coach Workspace Dashboard for managing client transformations and testimonials.
- High-Conversion Intake Funnel integrated with WhatsApp onboarding.
- Performance Optimized with Framer Motion animations and responsive design.

## 🛠️ Tech Stack

- React.js (Vite)
- Tailwind CSS
- Framer Motion
- Lucide React
- LocalStorage
